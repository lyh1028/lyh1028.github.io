无后效性

贪心：问题可以被分解成子问题，每个子问题的最优解总是能继续推出全局最优解

动态规划：问题可以被分解成子问题，已经求解的子问题不受后续阶段的影响。【无后效性】

无后效性：动态规划对状态空间的遍历构成一个有向无环图的拓扑序列，有向无环图的“节点”对应问题中的“状态”，边对应状态之间的“转移”，转移的选取是动态规划中的决策。

  • 如果之前的阶段求解的子问题的结果包含了一些不确定的信息，导致了后面的阶段求解的子问题无法得到，或者很难得到，这叫「有后效性」
  • 解决「有后效性」的办法是固定住需要分类讨论的地方，记录下更多的结果。
    • 例如，增加状态数组维数
    • 重新定义一个更细致的状态

环形子数组的最大和 lc918.

给定一个长度为 n 的环形整数数组 nums ，返回 nums 的非空 子数组 的最大可能和 。

环形数组 意味着数组的末端将会与开头相连呈环状。形式上， nums[i] 的下一个元素是 nums[(i + 1) % n] ， nums[i] 的前一个元素是 nums[(i - 1 + n) % n] 。

子数组 最多只能包含固定缓冲区 nums 中的每个元素一次。形式上，对于子数组 nums[i], nums[i + 1], ..., nums[j] ，不存在 i <= k1, k2 <= j 其中 k1 % n == k2 % n 。

简单思路

求环形子数组的最大和，即求中间某一段子序列的最小和，然后用总和-最小和得到ans1

将ans1与上题（lc53）方法求出的答案进行比较，取大者即可。

乘积最大子数组 lc152

给你一个整数数组 nums ，请你找出数组中乘积最大的非空连续子数组（该子数组中至少包含一个数字），并返回该子数组所对应的乘积。

测试用例的答案是一个 32-位 整数。

子数组 是数组的连续子序列。

容易按照lc53的思路想，并且注意到负负得正，所以维护一个最大值dp_max，维护一个最小值dp_min，当前遍历元素为a时，dp_max = max(dp_max*a, dp_min*a, a); dp_min = min(dp_min*a, dp_max*a, a);

01背包问题

有N个物体，每个物体的重量为W，价值为V，每种物体只能取一次，一个容量为M的背包能装的最大价值是多少？

状态方程：

优化：把dp数组由二维降维到一维，但此时要注意对背包容量的遍历一定要从大到小遍历。

有时候用一维数组就不会做错，用二维反而初始化很烦【初始化要注意dp[i][0]和dp[0][j]要取什么】

【对于完全背包，一定要掌握一维数组的做法，有些题不能用二维数组！】

注意，0-1背包不只有“物体”，其实本质上是一个集合，集合中的某个元素仅有 “取1个” 和 “不取” 两种选择，并且所取的东西要满足某种限制。

例如：从数组中取数，判断数组中是否有一个子集，子集元素之和是数组元素之和的一半。

对于“有几种取法”的问题，往往把上面的递推方程的max改成+，并对w和v做一些调整即可。

完全背包

有N个物体，每个物体的重量为W，价值为V，每种物体能取无数次，一个容量为M的背包能装的最大价值是多少？

二维动归方程：

该方程的推导可以看知乎，有严格的数学推导，用的是整体代入法。理解一下，因为第i个物体可取无数次，所以即使在[0-i]里选择了第i件，以后还是有机会选i，就不能用dp[i-1][j-w[i]]了。

一维动归方程：

只能用一维做的题：lc377. 组合总和 Ⅳ

求“排列”数，即{1,2,2}和{2,1,2}算作不同的方案，一定要外层遍历背包容量，内层遍历物品。

求“组合”数，即只要集合元素相同就是同一种方案，要外层遍历物品，内层遍历背包容量。

最长公共子序列

子列——不要求连续

可以把问题看做对t[i]和s[j]，分别选择取或不取

设dp[i][j] 表示数组t [0-i-1]和 s [0-j-1] 的最长公共子序列

当t[i]=s[j]时，dp[i+1][j+1] = dp[i][j] + 1

当t[i]!=s[j]时，两种选择：t[0-i]和s[0,j-1]的最长公共子序列；t[0,i-1]和s[0,j]的最长公共子序列。

dp[i+1][j+1] = max(dp[i][j+1],dp[i+1][j])

lc337 打家劫舍

小偷又发现了一个新的可行窃的地区。这个地区只有一个入口，我们称之为 root 。

除了 root 之外，每栋房子有且只有一个“父“房子与之相连。一番侦察之后，聪明的小偷意识到“这个地方的所有房屋的排列类似于一棵二叉树”。 如果 两个直接相连的房子在同一天晚上被打劫 ，房屋将自动报警。

给定二叉树的 root 。返回 在不触动警报的情况下 ，小偷能够盗取的最高金额 。

lc1039

你有一个凸的 n 边形，其每个顶点都有一个整数值。给定一个整数数组 values ，其中 values[i] 是第 i 个顶点的值（即 顺时针顺序 ）。

假设将多边形 剖分 为 n - 2 个三角形。对于每个三角形，该三角形的值是顶点标记的乘积，三角剖分的分数是进行三角剖分后所有 n - 2 个三角形的值之和。

返回 多边形进行三角剖分后可以得到的最低分 。

注意到我们在顶点0和顶点n-1之间选一点k，连接0和k，k和n-1，就能把该问题拆分为两个子问题 + 已经构成的三角形分数。即 \[ 选定i之后，f(0,n-1) = f(0,i) + f(i,n-1) + v[i]*v[n-1]*v[0] \] 区间DP写代码的时候有两种思路

  1. 遍历区间长度和区间起点
  1. 遍历区间起点和区间终点
